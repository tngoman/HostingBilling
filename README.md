# HostingBilling
Open source Web Hosting automation system with a built in content management system.

Addon modules included:

Control Panels
1. cPanel
2. CWP
3. Cyberpanel
4. Directadmin
5. ISP Config
6. Plesk

Payment Gateways
1. Bitcoin
2. 2Checkout
3. Coinpayments
4. Payfast
5. Paypal
6. Stripe
7. Razorpay
